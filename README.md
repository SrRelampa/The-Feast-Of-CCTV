# The Feast Of CCTV

Bienvenido a **The Feast Of CCTV**, una página web dedicada a un juego emocionante y entretenido.

## 📋 Contenido

- **Inicio** - Página principal con información general
- **Personajes** - Galería de personajes jugables
- **Power Ups** - Información sobre mejoras del juego
- **¿Cómo Jugar?** - Guía completa de instrucciones
- **Registro** - Formulario para registrarse como jugador

## 🎨 Diseño

El sitio utiliza una paleta de colores **lavanda** con gradientes modernos y un diseño responsivo que se adapta a cualquier dispositivo.

### Colores Principales:
- Lavanda Claro: #E6D4E8
- Lavanda Medio: #B38BC2
- Lavanda Fuerte: #6A4E9C
- Azul Lavanda: #A8C4E8
- Fondo: #F8F0FF

## 🚀 Cómo Usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. Personaliza el contenido según tus necesidades
4. Agrega tus imágenes en la carpeta `Img/`
